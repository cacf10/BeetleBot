1. To build ESP8266:
   Use ARDUINO IDE, choose the Board Type as "Generic ESP8266 Module". Build ESP8266 folder and download the program to ESP8266 Module.
   
3. Build Arduino
   USE ARDUINO IDE, choose the Board Type as "Arduino Nano"(Or your arduino type), Build "BeetleBot" folder and download to Arduino Board.
   
5. Remote Control
   After download the program to arduino, you can use a computer or a mobile device, such as phone or ipad, connect to the WIFI AP named "beetle robot", the password is "12345678".
   When wifi connected succesful, open your web browser such as Chrome, access the ip address "192.168.4.1", then you can open the remote control page, then you can control the robot.
